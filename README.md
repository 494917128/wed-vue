
# Vue-Wed-Dev

基于 Vue-cli，个人封装的组件/方法。

# 安装

#### NPM

``` bash
npm install vue-wed-dev --save
```

- [wed-modal](/docs/components/wed-modal.md) - 模态框组件
- [wed-toast](/docs/components/wed-toast.md) - 提示框组件
- [wed-loading](/docs/components/wed-loading.md) - loading框组件
